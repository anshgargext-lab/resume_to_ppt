# Resume-to-PPT System Features

This system enables the conversion of a resume into a concise one-slide professional PowerPoint presentation. The following features are enabled by this upload:

### Sections Discovered in the Resume

- Introduction
- Relevant Experience
- Projects
- Skills & Technology
- Education
- Certifications/Awards
- Overall Impression

### Output

The output is a concise one-slide professional PowerPoint presentation that summarizes the key points of the resume.

### Features and Limitations

- Certifications/Awards are only included if present in the resume.
- Education is mandatory if present in the resume.
- Experience and projects are selected based on content availability.
